# Audio

## Import Settings

### Music
- Format: Compressed (Vorbis/MP3)
- Load Type: Streaming
- Compression: Medium-High quality

### Sound Effects
- Format: PCM for short sounds
- Format: Compressed for longer sounds
- Load Type: Decompress on Load

## Folder Structure
- **Music/**: Background music tracks
- **SFX/**: Sound effects organized by category
- **Mixers/**: Audio mixer assets for volume control
