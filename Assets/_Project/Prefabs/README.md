# Prefabs

Store all reusable game objects here, organized by category:

- **Core/**: Essential prefabs (GameManager, AudioManager, etc.)
- **Player/**: Player character prefab
- **Enemies/**: All enemy prefabs
- **Environment/**: Level decoration and platforms
- **Items/**: Collectibles and power-ups
- **UI/**: UI canvas prefabs
- **VFX/**: Visual effects and particles
- **Projectiles/**: Bullets, arrows, etc.

## Naming Convention
- Use descriptive names: \`Enemy_Flying\`, \`Platform_Moving\`
- Prefix variants: \`Player_Blue\`, \`Player_Red\`
