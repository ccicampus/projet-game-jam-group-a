# Sprites

Organize all 2D artwork by category.

## Import Settings
- **Pixel Art**: Texture Type: Sprite, Filter Mode: Point, Compression: None
- **Smooth Art**: Texture Type: Sprite, Filter Mode: Bilinear
- Set consistent Pixels Per Unit (16, 32, or 64 for pixel art)

## Folder Organization
- **Characters/**: Player and enemy sprites with animation frames
- **Environment/**: Tilesets, platforms, backgrounds, props
- **Items/**: Collectibles, power-ups, pickups
- **UI/**: Buttons, icons, panels
- **VFX/**: Explosion frames, particle sprites
